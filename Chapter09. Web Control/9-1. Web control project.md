# 웹 브라우저로 제어하기

## 학습 목표

- Flask 서버가 HTTP 요청을 받아 MQTT 메시지로 변환하는 흐름을 설명할 수 있다.
- HTML 폼과 색상 선택기로 간단한 웹 제어 UI를 만들 수 있다.
- 웹 브라우저 → Flask → MQTT → TiCLE Lite 경로로 LED를 제어할 수 있다.

---

## 이 장의 도전 문제

> 웹 브라우저에서 색을 고르면, TiCLE Lite의 LED 화면이 즉시 바뀌도록 만들어 보자.

브라우저는 리모컨이고, Flask는 번역기이며, TiCLE Lite는 실행 장치이다. 각 역할을 명확히 구분하고 연결하는 것이 이 장의 핵심이다.

---

## 시스템 구조

```
웹 브라우저  →  Flask(PC)  →  MQTT 브로커  →  TiCLE Lite
  색상 선택      색상 변환      메시지 전달      LED 표시
```

## 미션 1. HTML 폼에서 색상을 받는 서버를 만들어라

### 문제

브라우저에서 색을 선택하고 "전송"을 누르면, 선택한 색상의 HEX 코드를 서버가 출력하게 만들어라.

### 생각해 보기

HTML `<input type="color">`는 색상 선택기를 제공한다. 폼 데이터는 POST 방식으로 서버에 전달된다.

### 코드

파일 이름: `m1_color_form.py` (PC에서 실행)

```python
# m1_color_form.py  — PC에서 실행
from flask import Flask, request

app = Flask(__name__)

HTML = """
<!DOCTYPE html>
<html>
<head><meta charset="utf-8"><title>색상 선택</title></head>
<body>
  <h2>LED 색상 선택</h2>
  <form action="/set" method="post">
    <input type="color" name="color" value="#00ff88">
    <button type="submit">전송</button>
  </form>
</body>
</html>
"""

@app.route("/")
def index():
    return HTML

@app.route("/set", methods=["POST"])
def set_color():
    color = request.form.get("color", "#000000")
    print("수신한 색상:", color)
    return f"<p>설정됨: {color} <a href='/'>뒤로</a></p>"

if __name__ == "__main__":
    app.run(host="0.0.0.0", port=5000, debug=True)
```

```sh
python m1_color_form.py
```

브라우저에서 `http://localhost:5000` 으로 접속한다.

### 관찰하고 실험해 보자

- 브라우저에서 색상을 선택해 전송하면 터미널에 HEX 코드가 출력되는가?
- HEX 코드 `#ff8800` 에서 빨강/초록/파랑 값은 각각 얼마인가?
- 다른 컴퓨터나 스마트폰에서도 서버에 접속할 수 있는가? (같은 Wi-Fi라면 가능하다)

### 알게 된 것

`#ff8800`에서 두 자리씩 16진수로 읽으면 RGB가 된다: `ff=255`, `88=136`, `00=0`. Python에서는 `int("ff", 16)` 으로 변환한다.

## 미션 2. HEX 색상을 RGB로 변환하고 MQTT로 발행하라

### 문제

HEX 코드를 파싱해 `(r, g, b)` 값으로 변환하고, `r,g,b` 형태의 문자열로 MQTT 발행하라.

### 코드

파일 이름: `m2_color_mqtt.py` (PC에서 실행)

```python
# m2_color_mqtt.py  — PC에서 실행
from flask import Flask, request
import paho.mqtt.client as mqtt

BROKER = "test.mosquitto.org"
TOPIC  = "ticle/web/color"

client = mqtt.Client()
client.connect(BROKER, 1883)

app = Flask(__name__)

HTML = """
<!DOCTYPE html>
<html>
<head><meta charset="utf-8"><title>LED 제어</title></head>
<body>
  <h2>LED 색상 제어</h2>
  <form action="/set" method="post">
    <input type="color" name="color" value="#00ff00">
    <button type="submit">적용</button>
  </form>
</body>
</html>
"""

def hex_to_rgb(hex_str):
    """#rrggbb 형식의 문자열을 (r, g, b) 정수 튜플로 변환한다."""
    h = hex_str.lstrip("#")
    r = int(h[0:2], 16)
    g = int(h[2:4], 16)
    b = int(h[4:6], 16)
    return r, g, b

@app.route("/")
def index():
    return HTML

@app.route("/set", methods=["POST"])
def set_color():
    color   = request.form.get("color", "#000000")
    r, g, b = hex_to_rgb(color)
    msg     = f"{r},{g},{b}"
    client.publish(TOPIC, msg)
    print(f"발행: {msg}")
    return f"<p>전송됨: rgb({r}, {g}, {b}) <a href='/'>뒤로</a></p>"

if __name__ == "__main__":
    app.run(host="0.0.0.0", port=5000, debug=False)
```


## 미션 3. TiCLE Lite가 색상 명령을 받아 LED 전체를 칠하게 하라

### 문제

`r,g,b` 형태의 MQTT 메시지를 수신해 LED 매트릭스 전체를 해당 색으로 채워라.

### 코드

파일 이름: `m3_led_color.py` (TiCLE Lite에서 실행)

```python
# m3_led_color.py
import network, utime
from ws2812 import WS2812Matrix
from umqtt.simple2 import MQTTClient

SSID   = "여기에 Wi-Fi 이름"
PW     = "여기에 Wi-Fi 비밀번호"
BROKER = "test.mosquitto.org"
TOPIC  = b"ticle/web/color"

sta = network.WLAN(network.STA_IF)
sta.active(True)
sta.connect(SSID, PW)
while not sta.isconnected():
    utime.sleep_ms(200)

m = WS2812Matrix([(0, 1)])

def on_message(topic, msg, retain, dup):
    try:
        parts = msg.decode().split(",")
        r, g, b = int(parts[0]), int(parts[1]), int(parts[2])
        # 밝기 제한 (LED 보호)
        r = min(r, 150)
        g = min(g, 150)
        b = min(b, 150)
        for x in range(16):
            for y in range(16):
                m[x, y].value = (r, g, b)
        m.update()
        print(f"적용: rgb({r},{g},{b})")
    except Exception as e:
        print("파싱 오류:", e)

client = MQTTClient(b"ticle-web", BROKER)
client.set_callback(on_message)
client.connect()
client.subscribe(TOPIC)
print("색상 명령 대기 중")

while True:
    client.check_msg()
    utime.sleep_ms(50)
```

```sh
replx m3_led_color.py
```

## 최종 프로젝트. 색상 팔레트 웹 리모컨

### 추가 기능

- 색상 선택기 외에도 사전 정의된 색상 버튼(빨강, 초록, 파랑, 끄기)을 추가한다.
- 현재 설정된 색을 화면에 미리보기로 표시한다.

### Flask 서버 (최종)

파일 이름: `web_server.py` (PC에서 실행)

```python
# web_server.py  — PC에서 실행
from flask import Flask, request
import paho.mqtt.client as mqtt

BROKER = "test.mosquitto.org"
TOPIC  = "ticle/web/color"

client = mqtt.Client()
client.connect(BROKER, 1883)

app = Flask(__name__)

PRESETS = {
    "red":   "255,0,0",
    "green": "0,255,0",
    "blue":  "0,0,255",
    "off":   "0,0,0",
}

HTML = """
<!DOCTYPE html>
<html>
<head>
  <meta charset="utf-8">
  <title>TiCLE 웹 리모컨</title>
  <style>
    body {{ font-family: sans-serif; text-align: center; padding: 30px; }}
    button {{ font-size: 18px; margin: 8px; padding: 10px 24px; cursor: pointer; }}
    .preview {{ width: 120px; height: 120px; margin: 20px auto;
                border: 2px solid #ccc; background: {preview}; }}
  </style>
</head>
<body>
  <h2>LED 색상 리모컨</h2>
  <div class="preview"></div>
  <form method="post" action="/preset">
    <button name="name" value="red"   style="background:#ff0000;color:#fff">빨강</button>
    <button name="name" value="green" style="background:#00cc00;color:#fff">초록</button>
    <button name="name" value="blue"  style="background:#0000ff;color:#fff">파랑</button>
    <button name="name" value="off"                             >끄기</button>
  </form>
  <hr>
  <form method="post" action="/custom">
    <input type="color" name="color" value="#00ff88">
    <button type="submit">직접 선택</button>
  </form>
</body>
</html>
"""

current_color = "#000000"

def hex_to_rgb(h):
    h = h.lstrip("#")
    return int(h[0:2],16), int(h[2:4],16), int(h[4:6],16)

@app.route("/")
def index():
    return HTML.format(preview=current_color)

@app.route("/preset", methods=["POST"])
def preset():
    global current_color
    name = request.form.get("name", "off")
    msg  = PRESETS.get(name, "0,0,0")
    client.publish(TOPIC, msg)
    r, g, b       = [int(x) for x in msg.split(",")]
    current_color = f"rgb({r},{g},{b})"
    return index()

@app.route("/custom", methods=["POST"])
def custom():
    global current_color
    color = request.form.get("color", "#000000")
    r, g, b = hex_to_rgb(color)
    client.publish(TOPIC, f"{r},{g},{b}")
    current_color = color
    return index()

if __name__ == "__main__":
    app.run(host="0.0.0.0", port=5000, debug=False)
```

```sh
python web_server.py
```

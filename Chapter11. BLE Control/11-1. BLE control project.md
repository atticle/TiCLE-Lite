# BLE UART 제어

## 학습 목표

- Nordic UART Service(NUS) 구조를 이해하고 BLE 시리얼 통신을 구현할 수 있다.
- 스마트폰 앱에서 문자열 명령을 보내 LED 패턴을 제어할 수 있다.
- 여러 명령 체계(색상, 패턴, 밝기)를 하나의 프로토콜로 설계할 수 있다.

---

## 이 장의 도전 문제

> 인터넷 없이, Wi-Fi 설정 없이, 스마트폰 앱 하나로 TiCLE Lite LED 화면의 색상과 패턴을 제어할 수 있을까?

BLE UART는 블루투스를 통해 마치 시리얼 통신처럼 문자열을 주고받는 방법이다. Chapter03(BLE)에서 배운 Write/Notify를 응용한다.

---

## 사전 지식. Nordic UART Service(NUS)

NUS는 BLE에서 텍스트를 주고받는 표준화된 서비스이다. TX와 RX 두 가지 특성으로 구성된다.

| 특성 | 방향 | 용도 |
|------|------|------|
| RX (쓰기) | 폰 → 보드 | 앱에서 명령 전송 |
| TX (알림) | 보드 → 폰 | 보드에서 응답 전송 |

**nRF UART** 앱 (iOS/Android, Nordic Semiconductor)을 사용하면 NUS와 쉽게 통신할 수 있다.

## 미션 1. NUS 서비스를 만들고 수신된 문자열을 출력하라

### 문제

NUS RX 특성에 Write된 문자열을 터미널에 출력하라.

### 코드

파일 이름: `m1_nus_rx.py`

```python
# m1_nus_rx.py
import bluetooth, utime
from micropython import const

_FLAG_WRITE = const(0x0008)

# NUS UUID (Nordic 표준)
_NUS_SVC_UUID = bluetooth.UUID("6E400001-B5A3-F393-E0A9-E50E24DCCA9E")
_NUS_RX_UUID  = bluetooth.UUID("6E400002-B5A3-F393-E0A9-E50E24DCCA9E")

_SERVICE = (_NUS_SVC_UUID, [(_NUS_RX_UUID, _FLAG_WRITE)])

ble = bluetooth.BLE()
ble.active(True)
((rx_handle,),) = ble.gatts_register_services((_SERVICE,))

name     = "TiCLE-UART"
name_enc = name.encode()
adv_data = bytes([2,0x01,0x06]) + bytes([1+len(name_enc),0x09]) + name_enc
ble.gap_advertise(100_000, adv_data)

def ble_irq(event, data):
    if event == 3:   # WRITE
        msg = ble.gatts_read(rx_handle).decode().strip()
        print("수신:", msg)

ble.irq(ble_irq)
print("nRF UART 앱으로 연결 후 문자를 보내 보자")

while True:
    utime.sleep_ms(100)
```


## 미션 2. 명령 체계를 설계하라

### 문제

다음 명령 문자열을 처리하는 파서를 만들어라.

| 명령 예시 | 동작 |
|-----------|------|
| `color:255,0,0` | 전체 빨간색 |
| `color:0,255,0` | 전체 초록색 |
| `bright:50` | 전체 밝기를 50으로 |
| `clear` | 화면 끄기 |

### 생각해 보기

명령 문자열을 `:` 로 나누면 명령 종류와 인자를 분리할 수 있다.

```python
msg   = "color:255,0,0"
parts = msg.split(":")
cmd   = parts[0]           # "color"
arg   = parts[1] if len(parts) > 1 else ""  # "255,0,0"
```

### 코드

파일 이름: `m2_command_parser.py`

```python
# m2_command_parser.py  (파서 로직만 테스트)

def parse_command(msg):
    """명령 문자열을 파싱해 (cmd, args) 를 반환한다."""
    parts = msg.strip().split(":", 1)
    cmd   = parts[0].lower()
    arg   = parts[1] if len(parts) > 1 else ""
    return cmd, arg

# 테스트
tests = ["color:255,0,0", "bright:80", "clear", "unknown:xyz"]

for t in tests:
    cmd, arg = parse_command(t)
    print(f"명령: '{cmd}'  인자: '{arg}'")
```

## 미션 3. 수신한 명령으로 LED를 제어하라

### 문제

NUS로 수신한 명령 문자열을 파싱해 `color:r,g,b`는 색상을, `clear`는 화면 끄기를 실행하라.

### 코드

파일 이름: `m3_ble_led.py`

```python
# m3_ble_led.py
import bluetooth, utime
from ws2812 import WS2812Matrix
from micropython import const

_FLAG_WRITE   = const(0x0008)
_NUS_SVC_UUID = bluetooth.UUID("6E400001-B5A3-F393-E0A9-E50E24DCCA9E")
_NUS_RX_UUID  = bluetooth.UUID("6E400002-B5A3-F393-E0A9-E50E24DCCA9E")
_SERVICE      = (_NUS_SVC_UUID, [(_NUS_RX_UUID, _FLAG_WRITE)])

ble = bluetooth.BLE()
ble.active(True)
((rx_handle,),) = ble.gatts_register_services((_SERVICE,))

m = WS2812Matrix([(0, 1)])

def fill(r, g, b):
    for x in range(16):
        for y in range(16):
            m[x, y].value = (r, g, b)
    m.update()

def handle_command(msg):
    parts = msg.strip().split(":", 1)
    cmd   = parts[0].lower()
    arg   = parts[1] if len(parts) > 1 else ""

    if cmd == "color":
        try:
            r, g, b = [min(int(v), 150) for v in arg.split(",")]
            fill(r, g, b)
            print(f"색상: rgb({r},{g},{b})")
        except Exception:
            print("color 인자 오류:", arg)

    elif cmd == "bright":
        try:
            val = min(int(arg), 150)
            fill(val, val, val)
            print("밝기:", val)
        except Exception:
            print("bright 인자 오류:", arg)

    elif cmd == "clear":
        m.clear()
        m.update()
        print("화면 꺼짐")

    else:
        print("알 수 없는 명령:", cmd)

def ble_irq(event, data):
    if event == 3:
        msg = ble.gatts_read(rx_handle).decode()
        handle_command(msg)

ble.irq(ble_irq)

name     = "TiCLE-UART"
name_enc = name.encode()
adv_data = bytes([2,0x01,0x06]) + bytes([1+len(name_enc),0x09]) + name_enc
ble.gap_advertise(100_000, adv_data)
print("BLE LED 제어 준비 완료")

while True:
    utime.sleep_ms(100)
```


## 최종 프로젝트. BLE UART로 LED 패턴 제어

### 추가 명령 확장

| 명령 | 동작 |
|------|------|
| `pattern:rainbow` | 무지개 패턴 |
| `pattern:snake` | 뱀(한 줄씩 순서 점등) |
| `color:r,g,b` | 단색 |
| `clear` | 끄기 |

### 최종 코드

파일 이름: `ble_uart.py`

```python
# ble_uart.py
import bluetooth, utime, math
from ws2812 import WS2812Matrix
from micropython import const

_FLAG_WRITE   = const(0x0008)
_NUS_SVC_UUID = bluetooth.UUID("6E400001-B5A3-F393-E0A9-E50E24DCCA9E")
_NUS_RX_UUID  = bluetooth.UUID("6E400002-B5A3-F393-E0A9-E50E24DCCA9E")
_SERVICE      = (_NUS_SVC_UUID, [(_NUS_RX_UUID, _FLAG_WRITE)])

ble = bluetooth.BLE()
ble.active(True)
((rx_handle,),) = ble.gatts_register_services((_SERVICE,))

m = WS2812Matrix([(0, 1)])

def hsv_to_rgb(h, s, v):
    """h:0~360, s:0~1, v:0~1 → (r, g, b) 0~255"""
    h = h % 360
    c = v * s
    x = c * (1 - abs((h / 60) % 2 - 1))
    r1, g1, b1 = [(c,x,0),(x,c,0),(0,c,x),(0,x,c),(x,0,c),(c,0,x)][int(h/60)]
    m_ = v - c
    return int((r1+m_)*255), int((g1+m_)*255), int((b1+m_)*255)

def fill(r, g, b):
    for x in range(16):
        for y in range(16):
            m[x, y].value = (r, g, b)
    m.update()

def pattern_rainbow():
    for step in range(36):
        for x in range(16):
            for y in range(16):
                hue = (x * 22 + step * 10) % 360
                r, g, b = hsv_to_rgb(hue, 1.0, 0.4)
                m[x, y].value = (r, g, b)
        m.update()
        utime.sleep_ms(50)

def pattern_snake():
    for i in range(256):
        x = (i % 16) if (i // 16) % 2 == 0 else (15 - i % 16)
        y = i // 16
        m.clear()
        m[x, y].value = (0, 200, 50)
        m.update()
        utime.sleep_ms(30)

cmd_queue = []

def ble_irq(event, data):
    if event == 3:
        msg = ble.gatts_read(rx_handle).decode().strip()
        cmd_queue.append(msg)

ble.irq(ble_irq)

name     = "TiCLE-UART"
name_enc = name.encode()
adv_data = bytes([2,0x01,0x06]) + bytes([1+len(name_enc),0x09]) + name_enc
ble.gap_advertise(100_000, adv_data)
print("BLE UART 준비 완료")

while True:
    if cmd_queue:
        msg   = cmd_queue.pop(0)
        parts = msg.split(":", 1)
        cmd   = parts[0].lower()
        arg   = parts[1] if len(parts) > 1 else ""

        print("명령:", cmd, "인자:", arg)

        if cmd == "color":
            try:
                r, g, b = [min(int(v), 150) for v in arg.split(",")]
                fill(r, g, b)
            except Exception:
                pass
        elif cmd == "clear":
            m.clear(); m.update()
        elif cmd == "pattern":
            if arg == "rainbow":
                pattern_rainbow()
            elif arg == "snake":
                pattern_snake()
        elif cmd == "bright":
            try:
                v = min(int(arg), 150)
                fill(v, v, v)
            except Exception:
                pass

    utime.sleep_ms(20)
```

```sh
replx ble_uart.py
```
